📋 Lesson Summary

1
Particles: Korean attaches particles to words like I and You to show what the sentence is about.
2
Topic marker: 는 — one of the most common particles (topic marker).
3
Structure: 너는 + description — “You” + compliment.
4
Pattern: The beginning 너는 stays the same — only the description changes.
5
Compliments: 최고야, 대단해, 아름다워 — you are the best / amazing / beautiful.

The core pattern

너는+description

너는 = You (as for you / when it comes to you)

Key particle

는 topic marker
너는
너는 = You
